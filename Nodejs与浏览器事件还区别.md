# Node与浏览器事件环区别
  ## 1. 任务队列数不同
  ## 2. Nodejs微任务执行时机不同
  ## 3. 微任务优先级不同

  >### 任务队列数
  + 浏览器中只有两个任务队列
  + Nodejs中有6个事件队列

  >### 微任务执行时机
  * 二者都会在同步代码执行完毕后执行微任务
  * 浏览器平台下每当一个宏任务执行完毕后，就会清空微任务
  * Nodejs平台在事件队列切换时，就会清空微任务

  >### 微任务优先级
  * 浏览器事件环中，微任务存放于事件队列，先进先出
  * Nodejs中process.nextTick优先于promise.then
